# Brew Brava — лендинг на Next.js

Современный одностраничный сайт для крафтовой пивоварни Brew Brava (Нячанг). Стек: **Next.js (App Router) + TypeScript + Tailwind CSS**.

## Запуск проекта

```bash
npm install
npm run dev
```

Приложение поднимется на http://localhost:3000.

## Структура
- `app/` — страницы и глобальный лэйаут.
- `components/` — UI-компоненты (Header, Hero, Section, карточки сортов, форма, футер).
- `public/images/` — плейсхолдеры для фоновых изображений и фото. Замените на реальные фото пивоварни/команды/кафе.
- `public/favicon.svg` — временный фавикон.

## Настройка контента
- Тексты, списки и переводы (RU / EN / VI) лежат в `app/page.tsx` внутри объекта `translations`.
- Карточки сортов и остальные копирайты берутся из тех же переводов.
- Ссылки в хедере и CTA ведут на соответствующие секции через якоря.
- Форму B2B можно связать с бекендом: добавьте отправку в `components/ContactForm.tsx` вместо текущей заглушки с валидацией.
- Шрифты подключены через `next/font` (Inter + Oswald). Цвета и темы задаются в `tailwind.config.ts` и `app/globals.css`.

## Примечания по дизайну
- Тёмная тема, акцентный янтарный цвет (`#d38b26`), контрастная типографика.
- Секции визуально разделены линиями/градиентами, карточки имеют мягкие границы и тени.
- Используются иконки `lucide-react` для декоративных элементов и буллетов.
